# OOPSBannerApp

## Use Cases

- UC1: Printed "OOPS" using simple print statement.
- UC2: Rendered OOPS as Banner using asterisk and spaces with "+" operator. 
- UC3: Rendered OOPS as Banner using String.join().
- UC4: Rendered OOPS as Banner using Array_loop.
- UC5: Rendered OOPS as Banner using Inline Array initialization.
- UC6: Rendered OOPS as Banner using Static Functions.
- UC7: Rendered OOPS as Banner using InnerClass.
- UC8: Rendered OOPS as Banner using HashMap.
